# Training dataset catalogs

This directory contains machine-readable manifests for external training resources that Lumi can use as a starting point for multimodal and domain-specific learning.

## Included datasets

- **Open Images** — Google's large-scale image dataset for object detection, classification, segmentation, and multimodal grounding experiments.
- **Deep Blue data archive** — University of Michigan's research data repository for curated datasets, documentation, and download links.

## Source URLs

- https://storage.googleapis.com/openimages/web/download_v7.html
- https://deepblue.lib.umich.edu/data/concern/data_sets/gq67jr854

## Viewer

Open `public/dataset-viewer.html` in a browser or run the app and visit `/dataset-viewer.html` to browse the catalogs.

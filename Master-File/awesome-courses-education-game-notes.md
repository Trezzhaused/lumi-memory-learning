# Awesome CS Courses for education-game planning

Repository: https://github.com/prakhar1989/awesome-courses

## Why this matters

This repository is a curated list of free, high-quality computer science courses with lecture videos, lecture notes, assignments, readings, and exams. For an education game system, it is useful as a source of topic sequencing, challenge design, and curriculum structure.

## Practical use cases

- Build learning tracks for systems, programming, AI, machine learning, security, graphics, and theory.
- Convert university course modules into game missions, quizzes, and progression milestones.
- Reuse the course structure as a scaffold for educational NPCs, quest chains, and mastery checks.
- Create a fallback library of technical topics when the game needs deeper educational content.

## Suggested categories to surface in the game

- Systems and architecture: operating systems, computer architecture, databases
- Programming and theory: compilers, algorithms, discrete math, CS fundamentals
- Intelligence and data: AI, machine learning, statistics
- Applied areas: security, graphics, networking

## Notes for implementation

Keep the repository as a reference resource rather than a hard-coded curriculum. The best use is to map course topics to game learning goals, difficulty levels, and progression checkpoints.

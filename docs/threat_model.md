# LUMI Threat Model

## Red-Team Attacks

- Weight tampering
- Audit log forgery
- Prompt injection
- Data poisoning
- Resource exhaustion
- Escape attempts

## Blue-Team Defenses

- Hash-chained audit logs
- CI-enforced safety gates
- Allowlisted commands and sandbox boundaries
- Invariant checks with hard fail behavior
- Hardware attestation and signed evidence

# LUMI Security Policy

## Reporting Vulnerabilities

Please report security vulnerabilities to:
security@lumi-ai.dev

Do not disclose publicly until coordinated with maintainers.

## Supported Versions

We provide security updates for the latest major release.

## Disclosure Policy

We follow coordinated disclosure.
Researchers will receive credit unless anonymity is requested.

## Audit

LUMI undergoes formal safety verification and
third-party audit readiness preparation.
